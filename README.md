## Getting Started

### Prerequisites

First thing first, install <a href="https://www.oracle.com/java/technologies/downloads/">Java</a> and follow those instruction from that documentation

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ingwerludwig/binar-reservasi-bioskop.git
   ```
  
2. Install <a href="https://flywaydb.org/download/community">Flyway</a> as your computer type was

3. Import Maven Project from cloned Project in your IDE

4. Right Click pom.xml and click reload

5. Make sure you configured your Google OAuth credential's ClientID, Client Secret, redirect URI <a href="console.cloud.google.com/apis/">here</a> and other required database credential, match to yours!

6. Create Database on your local with name 'SpringAppBinarAcademy'

7.  Migrate your database
   ```sh
   flyway migrate
   ```
8. Run your Spring Boot Project!
